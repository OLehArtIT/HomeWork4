Вторая часть задания

Селекторы

Универсальные

соответствует любому HTML-элементу.

теговый селектор

позволяет задавать стили для всех элементов данного типа (тега)

классовый селектор

позволяет задавать стили для одного и более элементов с одинаковым именем класса, размещенных в разных местах страницы

селектор идентификатора

позволяет задавать стили для конкретного элемента (id должен быть уникальным!). Записывается он почти так же, как и класс, только в CSS вместо точки ставится символ решетки #

селектор потомка

позволяет задавать стили к элементам, расположенным внутри элемента-контейнера

дочерний селектор

позволяет задавать стили к элементам, которые являются прямыми потомками

соседний селектор

выбирает элемент, который находится непосредственно после указанного элемента, если у них общий родитель.

родственный селектор

выбирает все элементы, которые находятся после указанного элемента, если у них общий родитель

селектор атрибута

позволяет задавать стили к элементам на основе имени атрибута или значения атрибута
